# Информационная система кинотеатра — v2 (полная версия)

## Структура проекта

```
cinema_v2/
├── app.py              # Flask-приложение (все роуты)
├── main.py             # Создание БД
├── seed.py             # Заполнение тестовыми данными
├── requirements.txt
├── static/
│   └── style.css       # Тёмно-фиолетовая тема
└── templates/
    ├── base.html           # Базовый шаблон с навигацией
    ├── login.html          # Страница входа
    ├── index.html          # Главная — сеансы сегодня
    ├── movies.html         # Список фильмов + поиск
    ├── movie_form.html     # Форма добавления/редактирования фильма
    ├── schedule.html       # Расписание + фильтры
    ├── session_form.html   # Форма добавления/редактирования сеанса
    ├── tickets.html        # Список билетов + фильтры
    ├── ticket_sell.html    # Продажа билета со схемой мест
    ├── clients.html        # Список клиентов + поиск
    ├── client_detail.html  # Карточка клиента + история
    ├── client_form.html    # Форма добавления/редактирования клиента
    ├── halls.html          # Залы с заполняемостью
    └── stats.html          # Статистика
```

## Установка и запуск

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Создать и заполнить базу данных
python main.py
python seed.py

# 3. Запустить Flask
python app.py
```

Приложение: http://127.0.0.1:5000

## Функционал v2 (расширенный)

- **Авторизация** — вход по логину/паролю, разграничение прав по ролям
- **Фильмы** — список, поиск по названию, добавление, редактирование, удаление
- **Расписание** — фильтр по дате/жанру/формату, добавление, редактирование, удаление сеансов
- **Билеты** — список с фильтром по статусу и поиском клиента, продажа билета
  со схемой свободных мест (динамически загружается через fetch), смена статуса, удаление
- **Клиенты** — список с поиском, карточка клиента с историей посещений,
  добавление, редактирование, удаление
- **Залы** — просмотр с прогресс-баром заполняемости
- **Статистика** — выручка, топ фильмов, билеты по статусам
